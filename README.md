# analise-diamantes
Análise exploratória do dataset fornecido pela disciplina de "Análise de Dados" da Universidade Vila Velha.
A análise inclui exploração, visualização e interpretação dos fatores que influenciam o preço de diamantes.

Objetivo:
- Explorar o dataset
- Identificar o comportamento geral do preço
- Identificar as variáveis mais relevantes para explicar variações de valor
- Identificar as relações entre características como carat, color, cut e clarity
- Identificar possíveis padrões e insights úteis para a marca fictícia dos diamantes

Tecnologias Utilizadas
- Python 3.x
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

Principais Etapas da Análise
1. Carregamento e inspeção inicial
   - Entendimento da estrutura do dataset
   - Verificação de tipos de dados
2. Estatísticas Descritivas
   - Medidades de tendência central
   - Dispersão
   - Distribuição do preço e carat
3. Análise Visual
   - Confecção de gráficos como:
   - Histogramas
   - Boxplots
   - Gráfico de Barras
4. Identificação de Padrões
5. Conclusões
   - Resumo dos achados mais importantes
